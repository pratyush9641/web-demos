# Pratyush Web Design — Demo Websites

Demo websites for local businesses in Salt Lake, Kolkata. Each site is a single
self-contained HTML file with WhatsApp booking, a rule-based chatbot, and (for
some) a private appointment-manager dashboard — no backend, no monthly cost.

## Demos

| Business | Type | Folder |
|---|---|---|
| All About Smiles | Dental clinic | [`all-about-smiles-demo/`](all-about-smiles-demo/) |
| Srejoni Spa & Aromatherapy | Ladies salon & spa | [`srejoni-spa-demo/`](srejoni-spa-demo/) |
| Chemistry Cracker | Chemistry tuition | [`chemistry-cracker-demo/`](chemistry-cracker-demo/) |

Each folder has `index.html` (public site). Dental and salon folders also have
`admin.html` — an appointment manager (localStorage, demo PIN 1234).

## Notes

- Stock photos are placeholders (marked `NOTE FOR SETUP` in the code) — replace
  with real business photos before delivering to a client.
- Sample reviews/timetables are marked for replacement with real content.
- Built with plain HTML/CSS/JS — deployable on GitHub Pages, Netlify, or any
  static host.
